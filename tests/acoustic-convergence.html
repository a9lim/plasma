Linear acoustic wave (hydro) — convergence test

Pure Euler: B ≡ 0 everywhere, smooth periodic 1D acoustic wave on [0,1]², γ = 5/3, η = 0. Sweep amplitudes A ∈ {1e-2, 1e-3, 1e-4} × resolutions N ∈ {32, 64, 128, 256} (12 runs). After one period T = λ/c_s = √(3/5) ≈ 0.7746 the analytic solution equals the IC. L1 error averaged per cell per conserved component over all 8 conserved variables; B components stay machine-zero. The amplitude sweep tests whether the L1 floor at A=1e-3 is fp32 noise (floor scales with A) or a constant-coefficient bug (floor independent of A).

How to interpret the slope

Expected: slope ≈ 3 in L1 (limited by RK3 SSP). Slope ≈ 2 is evidence the PPM limiter is clipping smooth extrema (the Tóth 2000 result for primitive-limiter PPM on smooth hydro). Slope ≈ 1 on a smooth hydro test is a bug — either in HLLC, update-conserved-weighted, the sanitization clamp, or PPM Phase A. Companion data: if CPAW (MHD) shows slope ~1 but this test recovers ~3, the bug is MHD-side (HLLD 5-wave path, EMF recipe, CT B update). If both are ~1, the bug is hydro-side.
